# Проект «Гонки» — игра

## Что это
Браузерная игра. Сейчас в разработке самый первый экран — **выбор города**.

## Текущий экран: выбор города
- Чёрный полноэкранный фон.
- По центру — большой светящийся 3D-шар Земли (библиотека **cobe**).
- Шар можно вращать: пальцем на телефоне, мышкой на компьютере.
- На шаре горят города-огоньки (голубые), которые пульсируют.
- Города: Лондон, Нью-Йорк, Токио, Сидней, Прага.
- При нажатии на огонёк внизу появляется название города.

## Технологии
- Одна HTML-страница (`index.html`), без сборки и без npm.
- Библиотека **cobe** подключается через CDN (esm.sh).
- Открывается прямо в браузере, работает на телефоне.

## Как запустить
Открыть `index.html` двойным щелчком в браузере.
(Если cobe не загрузится из-за ограничений `file://`, запустить локальный сервер:
`python -m http.server` и открыть `http://localhost:8000`.)

## Принципы
- Пользователь делает игру впервые — объяснять всё простыми словами, по-русски.
- Держать код в одном файле, пока проект маленький.
