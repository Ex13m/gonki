Как поедем?

Настоящий город — фотореалистичные 3D-дома от Google (те же, что в Google Земля). Для этого нужен бесплатный API-ключ Google.

Где взять ключ? (займёт минут 5)
  1. Зайди на console.cloud.google.com с аккаунтом Google.
  2. Создай проект (любое имя).
  3. В поиске найди Map Tiles API и нажми «Enable» (включить). Понадобится привязать платёжный аккаунт — у Google есть бесплатный лимит, для игры его хватает.
  4. Открой «APIs & Services → Credentials» и создай API key.
  5. Скопируй ключ и вставь его в поле выше. Ключ сохранится в твоём браузере.
